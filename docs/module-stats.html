(inner) testSkat(u, v, w, method) → {Array.<Number>}

Calculate SKAT test.

This function implements two methods for calculating the p-value from the SKAT test statistic:

  • Davies' approximation (method = davies). This method is a more exact approach, at the cost of performance.
  • Liu approximation (method = liu). This method matches the first 3 moments of the mixture chi-square distribution. It is faster, but has inflated type 1 error in the tails. It can also be used in the special edge case where the Davies' approximation yields a p-value of 0.
Parameters:
Name Type Default Description
u Array.<Number> Vector of score statistics (length m, number of variants).
v Array.<Number> Covariance matrix of score statistics (m x m).
w Array.<Number> Weight vector (length m, number of variants).
method String davies Can be "davies", or "liu". Davies is the method used in the original Wu et al. SKAT paper.
